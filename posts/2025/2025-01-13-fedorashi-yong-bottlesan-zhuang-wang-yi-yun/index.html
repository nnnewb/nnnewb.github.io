Fedora 使用 bottles 安装网易云

作于：2025-01-13 00:00 ，预计阅读时间 3 分钟

0. 前言

因为 Windows Hyper-V 动不动干掉一半内存，我反正是有点顶不住了。正好工作机需求比较单一，所以直接换了 Fedora 日常开发用。

家用环境尝试了一下 Fedora 能满足大部分场景，但 NTFS 兼容确实是有问题的。 外接的硬盘盒 RAID 1 + NTFS 大约 5TB 数据量，用 qBittorrent 检查完整性的时候挂载崩了。 有点担心写坏文件系统，所以家里还是继续用 Windows 然后干掉 Hyper-V 拉倒。 过年回来再考虑下做不做NAS。

说回正题。

bottles 是一个 wine 的包装，聚合了一些比较好用的功能比如快照、依赖管理。能省很多事。

但相对的，想正常用 bottles 就需要一些魔法上网的能力。

1. bottles 安装

这里选择使用 flatpak 版本的 Bottles 。

# 切换 flathub 镜像源
sudo flatpak remote-modify flathub --url=https://mirror.sjtu.edu.cn/flathub
# 安装 bottles
flatpak install flathub com.usebottles.bottles
# 启动并初始化 bottles，注意设置代理环境变量
flatpak run --env=http_proxy=http://192.168.7.220:7890 --env=https_proxy=http://192.168.7.220:7890 com.usebottles.bottles

启动后需要填写两个 ota 地址，填入官方文档的地址后点 Download，完成后点 Done 即可进入 Bottles 主界面。

2. 创建 Bottle

新建 Bottle，名称填中文或英文都可以。我填了网易云音乐。

环境选择应用程序。

其他都不改，点创建。

新建的 bottle 会自动装一部分必要的依赖，比如 gecko/mono/arial字体 之类，这一步也需要网络连接。

3. 安装依赖

点击刚创建好的 bottle，点击选项->依赖，安装这些依赖：

  • vcredist2022
  • webview2
  • riched20
  • cjkfonts

4. 安装网易云

下载最新版网易云安装包，用 bottles “运行可执行文件”功能启动，然后正常安装、启动。

启动后的网易云字体有问题，到设置里把字体改成“思源黑体 Medium”，然后重启网易云音乐即可。

现在大部分中文字体渲染应该没问题了，桌面歌词也可以使用。

5. 一些问题

我使用的 Fedora 41 KDE + Wayland 桌面下，网易云托盘图标是会有一点点黑边的。

网易云窗口缩放的时候有明显的渲染错误，但不影响使用。

部分地方的字体渲染会有锯齿，不够清晰。

6. 总结

总之能用。
